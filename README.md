# Инструкция по кастомизации и изменению внешнего вида сайта

Этот файл содержит руководство по быстрому изменению визуального стиля сайта без изменения логики работы бэкенда и структуры разметки. Пользуясь этой инструкцией, вы сможете легко адаптировать внешний вид под новые требования прямо во время сдачи.

---

## 🎨 1. Изменение цветовой гаммы (Тема оформления)

Вся цветовая палитра сайта вынесена в глобальные CSS-переменные в начале файла стилей. Изменив их значения, вы мгновенно перекрасите весь сайт.

* **Файл:** `style.css`
* **Строки:** 33–42
* **Что менять:**
  ```css
  :root {
      --primary-blue: #007bff;     /* Базовый цвет (кнопки, активные ссылки, акценты) */
      --dark-blue: #0d47a1;        /* Темный акцент (шапка, подвал, заголовки секций) */
      --bg-gray: #ced4da;          /* Серый цвет для границ и неактивных элементов */
      --context-gray: #dee2e6;     /* Светло-серый фон для разделов и карточек */
      --white: #ffffff;            /* Цвет фона основных блоков и текста на темном фоне */
  }
  ```
  *Пример:* Чтобы сделать сайт в зеленой корпоративной гамме, замените `--primary-blue` на `#28a745` (зеленый), а `--dark-blue` на `#1e7e34` (темно-зеленый).

## ✍️ 2. Изменение шрифтов

Сайт использует локальный шрифт `Inter` для независимости от интернет-соединения.

* **Файл:** `style.css`
* **Строки:** 1–30 (подключение шрифтов через `@font-face`) и строка 47 (переменная шрифта).
* **Что менять:**
  Если вам нужно изменить шрифт на другой (например, Arial или Roboto):
  1. Перейдите на строку 47:
     ```css
     --font-main: 'Inter', sans-serif;
     ```
  2. Замените `'Inter'` на название системного шрифта, например:
     ```css
     --font-main: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
     ```

---

## 🖼️ 3. Изменение логотипа и названия в шапке

Логотип и текстовое название проекта выводятся в шапке на всех страницах динамически.

* **Файл:** `header.php`
* **Строки:** 13–18
* **Что менять:**
  * Изменить изображение логотипа: замените файл в `src="assets/20220922_logo.jpg"` на свой.
  * Изменить название сайта:
    ```html
    <span class="logo-text">Учусь<span>.РФ</span></span>
    ```
    Замените текст `Учусь` и `.РФ` на название вашего варианта.

## 🚀 4. Настройка главного экрана (Hero Section)

Главный баннер на главной странице задает основное настроение сайта.

* **Файл:** `index.php`
* **Строки:** 11–28
* **Что менять:**
  * Заголовок (строка 15): `<h1 class="hero-title">Учусь.РФ</h1>`
  * Подзаголовок (строки 16–18): Текст с описанием направлений обучения.
  * Изображение справа (строка 24): `<img src="assets/hero.png" ...>` (можно заменить на любой другой векторный или растровый рисунок из папки `assets`).

## 🌟 5. Изменение преимуществ (Advantages)

Секция с карточками достоинств платформы.

* **Файл:** `index.php`
* **Строки:** 31–59
* **Что менять:**
  Каждое преимущество оформлено в виде блока `.advantage-card`.
  * Иконка (строки 42, 49, 56): вы можете заменить картинки в тегах `src="assets/..."`.
  * Текст заголовков и описаний: измените содержимое тегов `<h3>` и `<p>` внутри карточек преимущества.

## 💬 6. Кастомизация отзывов слушателей

Слайдер с мнениями выпускников.

* **Файл:** `index.php`
* **Строки:** 99–137
* **Что менять:**
  Каждый отзыв находится внутри блока `.review-slide`.
  * Текст отзыва (строки 106, 120): текст внутри тега `<p class="review-text">`.
  * Автор (строки 107–113, 121–127): измените аватар автора `src="assets/..."`, его имя в `<h4>` и описание курса в `<p class="author-meta">`.

## 📞 7. Контакты и ссылки на соцсети в подвале

Информация в нижней части сайта.

* **Файл:** `footer.php`
* **Строки:** 12–47
* **Что менять:**
  * Ссылки на социальные сети (строки 12–16): укажите ваши адреса в атрибутах `href="#"` тегов `<a>`.
  * Список контактов (строки 30–46): телефон в теге `href="tel:..."`, почта в `href="mailto:..."` и физический адрес.
